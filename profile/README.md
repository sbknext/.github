# sbknext

Solo-dev AI infrastructure in three product lines.

## 🧠 Brain — AI memory & backend
- **[corebrain](https://github.com/sbknext/corebrain)** — Rust-native MCP server (mcp.sbknext.com)
- **[forge-client](https://github.com/sbknext/forge-client)** — Brain SDK (Python · Node · Rust)
- **[brain-dashboard](https://github.com/sbknext/brain-dashboard)** — skills / memory / todos UI
- **[echo-ai](https://github.com/sbknext/echo-ai)** — conversational AI frontend

## 🔨 Forge — developer toolkit
- **[forge-sutra](https://github.com/sbknext/forge-sutra)** — code structural graph
- **[forge-secure](https://github.com/sbknext/forge-secure)** — security scan (secrets · vulns · SAST)
- **[forge-harness](https://github.com/sbknext/forge-harness)** — model-agnostic LLM reliability
- **[forge-skills](https://github.com/sbknext/forge-skills)** — shareable AI modes & skills
- **[forge-site](https://github.com/sbknext/forge-site)** — landing site

## 🤖 Bots — social automation
- **[forge-social](https://github.com/sbknext/forge-social)** — safe-pace social posting
- **[forge-linkedin](https://github.com/sbknext/forge-linkedin)** — LinkedIn engagement

## 🔌 Standalone
- **[claude-fuse](https://github.com/sbknext/claude-fuse)** — local-first observability for Claude Code
